# react-weather-app

> Simple weather app - exercise task for Free Code Camp course

## Start

```sh
$ npm install
$ npm start
```

Also, you can configure server port using `PORT` environment variable.

```sh
$ PORT=5000 npm start
```

## Testing

```sh
$ npm test
```
